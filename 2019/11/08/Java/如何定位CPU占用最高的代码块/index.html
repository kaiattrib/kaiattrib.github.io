如何定位CPU占用最高的代码块?

1. 先利用top查看 占用较高的进程

2 。利用 top -Hp pid 或 ps H -p pid -o pid,tid,pmem,pcpu,time,comm。查看某个进程中的线程占用CPU的情况。

3. 利用jstack pid 对应某个线程，注意 jstack 中的 nid 是16进制的，注意转换。

4.可以在线程启动的时候，设置线程的name，方便快速定位。
